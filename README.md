# Bone Hunt
### Description
You play as a dog who is trying to collect all of his bones.
When you collect all the bones you move on to the next level.
### Controlls
* Move with WASD.
# Infinite DDR
### Description
A Dance Dance Revolution ripoff which goes on for infinity.
### Controlls
* Up = W
* Down = S
* Left = D
* Right = A
